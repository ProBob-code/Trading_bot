Trading Strategy

Daily Strategy (Swing Trading)

Confluence-based signal generation using multiple indicator agreement:

Indicator BUY Condition SELL Condition
MA44 (Custom) Price > MA44 Price < MA44
EMA Crossover EMA9 > EMA21 EMA9 < EMA21
Ichimoku Cloud Price above cloud Price below cloud
MACD MACD > Signal MACD < Signal
RSI (14) RSI > 50 RSI < 50
Supertrend Bullish Bearish
Signal Rule: Minimum 3 indicators must agree (confluence) before generating a signal. This reduces false signals and improves accuracy.
